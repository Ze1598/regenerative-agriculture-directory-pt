# Regenerative Agriculture Directory Portugal

A curated directory of regenerative agriculture projects, farms, and sustainable food systems in Portugal.

## Overview

This project aims to map and promote initiatives that are regenerating the soil, increasing biodiversity, and producing healthy food in Portugal. It provides an interactive map and a searchable list of projects, filtering by location and practices.

## Features

- **Interactive Map**: Visualize projects across Portugal.
- **Search & Filter**: Find projects by name, location, practice (e.g., Agroforestry, Holistic Grazing), or product.
- **Detailed Listings**: View contact info, practices, and products for each project.
- **Responsive Design**: Works on desktop and mobile.
- **SEO Optimized**: Built for visibility.

## Data

The dataset is stored in `data.js` and includes:
- Project Name
- Location (City, Coordinates)
- Contact Info (Phone, Email, Website)
- Regenerative Practices
- Products

## Tech Stack

- **HTML5**
- **CSS3** (Custom properties, Flexbox/Grid)
- **JavaScript** (Vanilla)
- **Leaflet.js** (Maps)
- **Font Awesome** (Icons)
